# Headings

HTML includes six levels of headings, which are ranked according to importance.

These are `**<h1>,<h2>,<h3>,<h4>,<h5>,**and**<h6>.**`

```html
<!DOCTYPE html>
<html>
<body>
  <h1>Heading 1</h1>
  <h2>Heading 2</h2>
  <h3>Heading 3</h3>
  <h4>Heading 4</h4>
  <h5>Heading 5</h5>
  <h6>Heading 6</h6>
</body>
</html>
```

output:

# This is heading 1

## This is heading 2

### This is heading 3

### This is heading 4

### This is heading 5

### This is heading 6
