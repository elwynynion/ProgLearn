# Lists

**Ordered Lists**

- An ordered list start with **<OL>** tag, and each list item is defined by **<LI>** tag.
- Each item on the list is organized through numbers. in increasing order:

example: 

```html
<html>
  <head>
    <title>Lists</title>
  </head>
<body>
  <ol>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
  </ol>
</body>
</html>
```

Output:

1. Coffee
2. Tea
3. Milk


**UNORDERED LISTS**

- An unordered lists starts with the `<UL>` tag; each item is also defined by the `<LI>` tag

example:

```html
<html>
 <head>
  <title>Lists</title>
 </head>
<body>
 <ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
</body>
</html>

Output:
- Coffee
- Tea
- Milk
```
